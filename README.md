## Featured Projects

### 🤖 AI & LLM Systems

**[Industrial Maintenance Agent](https://github.com/SatvikSPandey/industrial-maintenance-agent)** · [Live Demo](https://industrial-maintenance-agent-satvik.streamlit.app)  
5-node LangGraph agentic pipeline for autonomous fault detection and maintenance scheduling. Uses local Llama 3 via Ollama — no cloud API costs. State-machine architecture with tool-calling, self-correction loops, and structured JSON reasoning.  
`LangGraph` `LangChain` `Ollama` `Llama 3` `Streamlit`

**[Semantic Search Engine](https://github.com/SatvikSPandey/semantic-search-engine)** · [Live Demo](https://semantic-search-engine-satvik.streamlit.app)  
RAG-powered document search over ChromaDB with AWS Bedrock (Claude 3 Haiku) as the LLM backend. Sentence-transformer embeddings, graceful API fallback pattern, sub-second retrieval.  
`ChromaDB` `AWS Bedrock` `sentence-transformers` `RAG` `Streamlit`

**[Tender AI](https://github.com/SatvikSPandey/tender-ai)** · [Live Demo](https://tender-ai-satvik.streamlit.app)  
5-agent pipeline that extracts 39–42 out of 42 structured fields from government tender PDFs. Dockerized and deployable in one command.  
`LangChain` `Multi-Agent` `PDF Parsing` `Docker` `Streamlit`

**[WhisperDrop](https://github.com/SatvikSPandey/whisperdrop)** · [Live Demo](https://whisperdrop-satvik.streamlit.app)  
Drag-and-drop audio transcription using OpenAI Whisper via Groq API. Real-time streaming output, multi-format export (txt, srt, json), and speaker-optimised preprocessing.  
`Whisper` `Groq API` `Streamlit` `Audio Processing`
